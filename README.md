# reWASD configs

Controller configurations for reWASD

## Controllers

Includes configs for:

- Xbox Elite
- DualSense
- Virtual controller input via Moonlight/Parsec

Not all controller configs are available for all games.

Current games include:

- Cyberpunk 2077

- Forza Horizon 5
- Horizon Zero Dawn
- No Man's Sky
